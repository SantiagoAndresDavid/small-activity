# small-activity

Este repositorio contiene una serie de actividades pequeñas organizadas en diferentes ramas. Cada rama representa una actividad o proyecto específico. El propósito de este repositorio es gestionar y versionar cada actividad de manera independiente para facilitar su consulta y desarrollo.

## Estructura del Repositorio

Cada actividad está ubicada en una rama dedicada. Las actividades son independientes entre sí, permitiendo que cada rama contenga código, documentación y recursos relacionados específicamente con su actividad.

### Ramas Actuales

1. **audith-database**: Esta primera rama se enfoca en la auditoría de bases de datos. En ella se desarrollan scripts y herramientas relacionadas con la revisión y control de integridad, optimización de consultas, y la detección de inconsistencias en los datos. 

   - **Objetivos**:
     - Implementar auditorías para bases de datos de diferentes tipos.
     - Asegurar la integridad y consistencia de los datos.
     - Optimizar el rendimiento de las consultas y la estructura de las tablas.
   - **Recursos**:
     - Scripts SQL
     - Documentación sobre auditoría de bases de datos
